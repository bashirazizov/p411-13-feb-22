### Tapşırıq

Facebookun bazasını qurduğunuzu düşünün.

1. User table-i olsun. (Id, Name, Surname, ProfilePhoto, Biography, PostCount)

2. Posts table-i olsun. (Id, Image, Text)

3. Comment table-i oldun. (Id, Text)

4. User-lərin postları olsun. Postların comment-ləri olsun.

5. Bütün postları istifadəçiləri ilə birlikdə göstərən view yaradın.

6. İstifadəçi id-si qəbul edib, istifadəçinin postlarına olunan kommentlərin sayını qaytaran funksiya yazın.

7. Post əlavə olunanda User-in PostCount-u avtomatik artsın.

Execute etdiyiniz sql command-ları silməyin. Həmin sql kodları əsasında qiymətləndirmə aparacağam. Kodları 1 faylda düzgün sıra ilə yazın, push edin.
